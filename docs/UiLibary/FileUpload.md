# 大文件上传

## 功能设计

- 1.拖拽上传

  - `支持拖拽文件到指定区域上传`
  - `拖拽时显示高亮边框和提示信息`
  - `拖拽离开或放下时恢复默认状态`
  - `拖拽文件格式和大小校验`

- 2.多文件选择

  - `点击按钮触发文件选择对话框`
  - `支持多选文件`
  - `文件选择后自动添加到上传队列`
  - `文件数量和大小的校验`

- 3.分片上传

  - `大文件自动分片上传`
  - `每个分片单独上传和重试`
  - `支持暂停和恢复上传`
  - `支持并行上传多个分片`

- 4.断点续传

  - `上传前先校验文件(秒传)`
  - `记录已上传分片`
  - `断网恢复后继续上传未完成分片`
  - `基于文件hash标识文件唯一性`
