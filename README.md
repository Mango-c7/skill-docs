# mango-docs

## 项目介绍

mango-docs 是一个基于 VitePress 的文档项目，用于展示个人博客的技术分享。

欢迎大家提出宝贵意见，共同完善这个项目！

## Node 环境要求

- Node.js 版本 >= 22.16.0
- pnpm 版本 >= 10.0.0

项目 Node.js 22.17.0 和 pnpm 10.13.1 进行开发。

## 安装依赖

```bash
pnpm install
```

## 本地运行

```bash
pnpm run dev
```

## 构建静态文件

```bash
pnpm run build
```

## 预览静态文件

```bash
pnpm run preview
```

## 代码 commit 规范

```bash

git add .

pnpm run commit

git push github main
```
